📤 Output Settings

Requests JSON format from the model. Disable if model struggles with JSON.

If a segment keeps coming back malformed, translate it on its own as plain text (no JSON). Slower, but avoids leaving text untranslated or breaking the page.

    {%- elif message['role'] == 'assistant' -%}
        {{ '<start_of_turn>model\n' + message['content'] | trim + '<end_of_turn>\n' }}
